# LiTTree OS Brand Direction

LiTTree should feel like a video game that happens to be an AI operating system.

## Core Metaphor

- Roots: memory and long-term context.
- Branches: agents and specialized skills.
- Leaves: active tasks.
- Fruits: finished projects, deploys, media, and marketplace wins.
- Energy flow: AI thinking, routing, and learning.

## Visual Identity

- Void Black: `#050507`
- Electric Cyan: `#2DF6FF`
- Solar Orange: `#FF8A1C`
- Emerald Glow: `#49FF9E`
- White: `#F7FAFC`

Avoid generic purple SaaS dominance. Cyan plus orange is the signature.

## Product Feel

- LiTT is visible, expressive, and present across the product.
- The tree is not a logo; it is the OS map.
- Backgrounds should feel like living worlds: stars, circuit vines, fog, glass, and energy streams.
- Panels should feel like glass holograms, not static cards.
- Motion should communicate state: thinking, success, errors, deployments, idle, and memory recall.

## UX Rule

Never copy Google AI Studio, Cursor, Qwen, or generic chat layouts as the final identity. Those patterns can inform usability, but LiTTree should be recognized by:

- LiTT companion presence.
- The glowing LiTTree world.
- Roots/branches/leaves/fruits system language.
- Cyan/orange/emerald energy.
- Game-like missions, progress, unlocks, and celebrations.

## First Surfaces To Keep Branded

- Studio loading screens.
- Studio chat welcome state.
- Dashboard command center.
- Agent picker.
- Global assistant.
- Marketplace hero.
